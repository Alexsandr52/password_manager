# password_manager

You can create a password and store it in plain text.
main.py to start application
You can change certain parameters via a file 'password_manager/sattings.py' then run it to save changes
Password manager
            commands:
                [1]: generate new password
                [2]: to save password
                [3]: to watch all passwords
                [4]: to remove password
                [5]: help information
                [0]: to exit
